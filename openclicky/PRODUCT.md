# Done — Automate Anything

**Pitch:** Automate anything on your screen. It actually does it.

## Product job

An **outcome agent** that sees your screen, uses tools, and finishes the job — not a tutor, not a chat wrapper, not brittle click replay.

## Three intent inputs

| Input | How it starts |
|-------|----------------|
| **Say** | Double-tap Shift → type command (or voice) |
| **Show** | Demonstrate once → saved outcome playbook |
| **Watch** | Product detects repeat work → offers takeover |

All three feed the same execution engine.

## Success metric

**Outcomes completed per user per week** — verified on screen or disk, ending in a **Done.** notification.

## ICP (initial)

Ops, VAs, bookkeepers — people who repeat the same desktop workflows weekly.
